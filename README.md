# BNP CRA

## Installation

Run `npm install`.

## Usage

1. Download your "JS CRA 10000ft" Google Sheet file as CSV.
2. Set your contract number in the `settings.json` file.
3. Run `npm run pdf -- --input ../path/to/your/cra.csv`.
